## ИНСТРУКЦИЯ ПО ПОЛУЧЕНИЮ КОНКРЕТНЫХ ССЫЛОК НА ВИДЕО

### Для YouTube:

1. Перейдите по ссылке на канал (например, https://www.youtube.com/@BenFelixCSI)
2. На странице канала нажмите на иконку поиска (лупа)
3. Введите название темы из списка выше
4. Выберите подходящее видео
5. Скопируйте ссылку из адресной строки браузера (формат: https://www.youtube.com/watch?v=VIDEO_ID)
6. Для встраивания используйте формат: https://www.youtube.com/embed/VIDEO_ID

### Для Khan Academy:

1. Перейдите по ссылке на раздел (например, ссылки выше)
2. Найдите конкретное видео по теме
3. Скопируйте прямую ссылку на видео из адресной строки
4. Используйте эту ссылку напрямую для встраивания

### Для Coursera/edX:

1. Перейдите на страницу курса
2. Выберите "Audit Course" (бесплатный режим)
3. Найдите раздел с видео лекциями
4. Скопируйте прямую ссылку на видео или используйте embed код если доступен

## ПРИМЕЧАНИЕ

⚠️ **ВАЖНО**: Большинство ссылок выше указывают на каналы и разделы платформ, где нужно найти конкретные видео. Это сделано потому, что:

1. Конкретные видео могут быть удалены или перемещены
2. На каналах постоянно появляются новые видео
3. Лучше найти актуальное видео по теме, чем использовать устаревшую ссылку

**Рекомендация**: Используйте ссылки на каналы выше, найдите конкретные видео по темам, скопируйте их прямые ссылки и добавьте в `videoLibrary.ts` или `courseData.ts` в поле `videoUrl`.
