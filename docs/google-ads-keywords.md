# Google Ads Keywords Export Guide

## Обзор

Система экспорта ключевых слов для Google Ads с поддержкой:
- 4 языков: русский, английский, польский, немецкий
- Региональных вариантов: Германия, Австрия, Швейцария, Великобритания, Ирландия, Польша, Россия
- Разных типов соответствия: точное, фразовое, широкое с модификаторами
- Категорий: инвестиции, брокеры, форумы, образование, аналитика, новости
- Отрицательных ключевых слов

## Структура ключевых слов

### Типы соответствия (Match Types)

1. **Точное соответствие** `[keyword]` - показ только при точном совпадении
2. **Фразовое соответствие** `"keyword"` - показ при наличии фразы в запросе
3. **Широкое с модификаторами** `+keyword +word` - показ при наличии всех слов с +

### Приоритеты

- **High** - Высокий коммерческий интент (проверка брокеров, куда инвестировать)
- **Medium** - Средний интент (форумы, образование)
- **Low** - Низкий интент (новости, аналитика)

## Использование

### Экспорт всех ключевых слов

```bash
npx tsx scripts/export-google-ads-keywords.ts
```

Скрипт создаст папку `exports/google-ads-keywords/` с файлами:

- `keywords-{language}-{region}.csv` - все ключевые слова для языка и региона
- `keywords-{language}-{region}.txt` - текстовый формат (один ключ на строку)
- `keywords-{language}-{region}-{category}.csv` - ключевые слова по категориям
- `negative-keywords.csv` - отрицательные ключевые слова
- `summary.json` - сводка по экспорту

### Программное использование

```typescript
import { 
  getAllKeywordsForLanguage,
  getGoogleAdsKeywords,
  exportKeywordsToCSV,
  exportKeywordsToText,
  getNegativeKeywords,
} from '@/utils/googleAdsKeywords';

// Получить все ключевые слова для языка и типа страницы
const keywords = getAllKeywordsForLanguage('ru', 'home', 'RU');

// Получить ключевые слова по категории
const brokerKeywords = getGoogleAdsKeywords('en', 'broker', 'GB');

// Экспорт в CSV
const csv = exportKeywordsToCSV(keywords);

// Экспорт в текст
const text = exportKeywordsToText(keywords);

// Отрицательные ключевые слова
const negatives = getNegativeKeywords('ru', 'broker');
```

## Отрицательные ключевые слова

Используйте отрицательные ключевые слова для исключения нерелевантных запросов:

**Общие:**
- бесплатно, скачать, торрент, мошенники
- free, download, torrent, scam, fraud
- kostenlos, herunterladen, betrug
- darmowe, pobierz, oszustwo

**По категориям:**
- Investment: работа, вакансия, кредит, займ
- Broker: страховка, недвижимость
- Forum: знакомства, чат
- Education: университет, колледж

## Рекомендации по настройке кампаний

1. **Начните с высокоприоритетных групп:**
   - Проверка брокеров
   - Куда инвестировать

2. **Используйте разные типы соответствия:**
   - Точное соответствие для контроля бюджета
   - Фразовое для баланса охвата и контроля
   - Широкое с модификаторами для расширения охвата

3. **Настройте отрицательные ключевые слова:**
   - Добавьте на уровне кампании и группы объявлений
   - Регулярно обновляйте на основе поисковых запросов

4. **Создайте отдельные объявления для каждого типа соответствия:**
   - Разные заголовки и описания
   - Разные целевые страницы

5. **Мониторинг и оптимизация:**
   - Отслеживайте поисковые запросы
   - Добавляйте новые отрицательные ключевые слова
   - Удаляйте неэффективные ключевые слова

## Статистика

После экспорта проверьте файл `summary.json` для получения статистики:
- Общее количество ключевых слов
- Количество по языкам
- Дата экспорта

## Обновление ключевых слов

Ключевые слова обновляются в файле `src/utils/googleAdsKeywords.ts`. После изменений:

1. Запустите экспорт: `npx tsx scripts/export-google-ads-keywords.ts`
2. Импортируйте новые файлы в Google Ads
3. Обновите существующие кампании или создайте новые

## Поддержка

При добавлении новых языков или регионов:
1. Добавьте ключевые слова в соответствующие функции в `googleAdsKeywords.ts`
2. Обновите массивы `LANGUAGES` и `REGIONS` в скрипте экспорта
3. Запустите экспорт
