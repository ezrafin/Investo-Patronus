

# Предложения по улучшению сайта

Проанализировав текущее состояние проекта, вот направления, которые можно улучшить без затрагивания авторизации:

---

## 1. Удаление аккаунта (из утверждённого ранее плана)

В профиле добавить кнопку "Delete Account" с диалогом подтверждения. Реализация через edge function, которая деактивирует/удаляет пользователя.

- Новый файл: `supabase/functions/delete-account/index.ts`
- Изменение: `ProfilePage.tsx` -- добавить кнопку + AlertDialog
- Переводы: 7 файлов `ui.json`

---

## 2. SEO и мета-теги для информационных страниц

Страница Contact не использует `<SEOHead>` корректно (нет `title` в компоненте). About имеет хардкод английских мета-тегов вместо переведённых. Привести к единому стандарту с переводами.

- Изменения: `AboutPage.tsx`, `CareersPage.tsx`, `DisclaimerPage.tsx`, `PrivacyPage.tsx`, `TermsPage.tsx` -- использовать `t()` для SEO title/description

---

## 3. Улучшение 404-страницы

Текущая 404-страница минимальна -- только заголовок и ссылка "Back Home". Можно добавить:

- Поиск по сайту (интеграция с `GlobalSearch`)
- Популярные ссылки (Markets, News, Analytics)
- Визуальный элемент (иллюстрация или анимация)

- Изменение: `NotFound.tsx`

---

## 4. Scroll-to-top при навигации

При переходе между страницами через React Router страница не скроллится наверх. Добавить компонент `ScrollToTop`.

- Новый файл: `src/components/navigation/ScrollToTop.tsx`
- Изменение: `App.tsx` -- добавить `<ScrollToTop />` внутри `BrowserRouter`

---

## 5. Мобильная оптимизация главной страницы

Hero-секция занимает 80vh с видео-фоном, что тяжело для мобильных. Можно:

- Отключить автовоспроизведение видео на мобильных (показывать статичный фон)
- Уменьшить `min-h` для мобильных
- Оптимизировать размеры шрифтов hero-заголовка

- Изменение: `HeroSection.tsx`

---

## 6. Skeleton для EmailSubscription

Блок `EmailSubscription` на главной рендерится без Suspense-обёртки, в отличие от остальных секций. Для консистентности обернуть.

- Изменение: `Index.tsx`

---

Какие из этих улучшений вы хотите реализовать? Можно выбрать несколько или все сразу.
