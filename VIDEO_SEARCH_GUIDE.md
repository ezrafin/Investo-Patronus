# Руководство по поиску видеоматериалов для видеобиблиотеки и модулей обучения

## Общие рекомендации

### Приоритетные источники:
1. **Khan Academy** (Economics & Finance domain) - https://www.khanacademy.org/economics-finance-domain
2. **YouTube образовательные каналы**:
   - Ben Felix
   - The Plain Bagel
   - Two Cents
   - Investopedia
   - Financial Education
3. **Coursera** (audit mode - бесплатный доступ к видео)
4. **edX** (free audit)
5. **MIT OpenCourseWare**
6. **Yale Courses** (Financial Markets)

### Критерии отбора видео:
- **Язык**: English (можно с субтитрами)
- **Качество**: приемлемое качество звука и изображения
- **Формат**: Educational (не реклама, не торговые сигналы, не продвижение брокеров)
- **Длительность**: близка к указанной (±10 минут допуск)
- **Уровень**: соответствует указанному (Beginner/Intermediate/Pro)

### Исключить:
- Торговые сигналы и прогнозы
- Продвижение конкретных брокеров/платформ
- Слишком продвинутый уровень для beginner тем
- Низкое качество записи

## Процесс поиска для каждой темы

1. **Формулировка поискового запроса**
   - Основные ключевые слова из названия темы
   - Добавить "tutorial", "explained", "beginner" или "educational"
   - Указать длительность если важно

2. **Проверка источников**
   - Начать с Khan Academy для базовых тем
   - Проверить рекомендованные YouTube каналы
   - Рассмотреть Coursera/edX для структурированных курсов

3. **Оценка видео**
   - Соответствие теме
   - Длительность
   - Качество (звук, изображение)
   - Образовательный характер (не реклама)
   - Уровень сложности

4. **Сохранение ссылок**
   - YouTube: использовать embed URL формат
   - Khan Academy: использовать прямую ссылку на видео
   - Другие платформы: проверить возможность встраивания
